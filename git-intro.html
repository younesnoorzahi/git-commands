مقدمات گیت و نصب آن

گیت چیست؟

گیت (Git) یه سیستم کنترل نسخه خیلی معروف و پرکاربرده که برنامه‌نویس‌ها و تیم‌ها ازش استفاده می‌کنن تا کدهاشون رو مدیریت کنن. با گیت می‌تونی تغییرات فایل‌هات رو ذخیره کنی، به نسخه‌های قبلی برگردی، با چند نفر همزمان روی یه پروژه کار کنی و همه چیز رو مرتب و منظم نگه داری.

چرا گیت؟

قبل از گیت، مدیریت تغییرات کدها سخت بود و ممکن بود نسخه‌های مختلف بهم بخورن یا اطلاعات از دست بره. اما گیت این مشکلات رو حل کرده و کار باهاش ساده و سریع شده.

نصب گیت

برای نصب گیت، کافیه به سایت رسمی گیت به آدرس git-scm.com بری و نسخه مناسب سیستم‌عاملت رو دانلود و نصب کنی. گیت روی ویندوز، مک و لینوکس قابل نصب و استفاده است.

مراحل نصب روی ویندوز:

  1. وارد سایت git-scm.com شو.
  2. روی دکمه دانلود ویندوز کلیک کن.
  3. فایل دانلود شده رو اجرا کن و مراحل نصب رو پیش برو (می‌تونی تنظیمات پیش‌فرض رو قبول کنی).
  4. بعد از نصب، ترمینال Git Bash رو باز کن تا دستورات گیت رو اجرا کنی.

نصب روی مک:

اگر مک داری، ساده‌ترین راه استفاده از Homebrew هست. ترمینال رو باز کن و این دستور رو بزن:

brew install git

نصب روی لینوکس:

بسته به توزیعت، مثلا اوبونتو:

sudo apt update
sudo apt install git

بعد از نصب

برای اطمینان از نصب درست، تو ترمینال بنویس:

git --version

اگر نسخه گیت نمایش داده شد یعنی نصب موفق بوده.

تنظیمات اولیه گیت

برای اینکه گیت بدونه کی هستی و تغییرات رو به چه کسی نسبت بده، باید نام و ایمیل خودتو تنظیم کنی:

git config --global user.name "نام شما"
git config --global user.email "ایمیل شما"
