🚀 شروع پروژه با Git

برای شروع کار با گیت در یک پروژه، فقط کافیه مراحل زیر رو طی کنی: 


cd my-project
git init
git status
git add .
git commit -m "اولین کامیت: شروع پروژه"

🔹 git init: گیت رو در پروژه فعال می‌کنه
🔹 git add .: همه‌ی فایل‌ها رو برای commit آماده می‌کنه
🔹 git commit: اولین ذخیره در تاریخچه گیت انجام میشه
