آشنایی و استفاده از GitHub

🔍 GitHub چیست؟

GitHub یک پلتفرم محبوب برای میزبانی کد، همکاری تیمی و مدیریت پروژه است که بر پایه Git کار می‌کند.

🚀 شروع سریع

  1. ثبت‌نام در github.com
  2. ساخت Repository جدید
  3. اتصال پروژه خود به مخزن GitHub

🔧 اتصال پروژه به GitHub

git init
git add .
git commit -m "اولین کامیت"
git remote add origin https://github.com/username/repo.git
git push -u origin main

📥 کلون کردن پروژه

git clone https://github.com/username/repository.git

📦 GitHub Pages

GitHub این امکان را می‌دهد تا سایت استاتیک خود را به رایگان روی اینترنت منتشر کنید.

مسیر: Settings → Pages → انتخاب branch → آدرس: https://username.github.io/repo/

📌 سایر امکانات مفید

  • Issues: گزارش باگ و پیگیری مشکلات
  • Pull Requests: پیشنهاد تغییرات در کد
  • Actions: اجرای خودکار تست و دیپلوی
  • Projects: مدیریت وظایف به سبک کانبان
  • Wiki: مستندسازی پروژه
