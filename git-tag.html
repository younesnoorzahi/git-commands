تگ زدن در Git

🏷️ تگ چیست؟

تگ‌ها برای نشانه‌گذاری نسخه‌های خاصی از تاریخچه پروژه استفاده می‌شن؛ معمولاً برای انتشار نسخه‌ها مثل v1.0.0.

🛠️ ساخت تگ

git tag v1.0.0

🚀 ارسال تگ به ریموت

git push origin v1.0.0

📜 نمایش همه تگ‌ها

git tag

🧹 حذف تگ

حذف تگ محلی:

git tag -d v1.0.0

حذف تگ از ریموت:

git push origin --delete v1.0.0
